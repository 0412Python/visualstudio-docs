# Math.fround Function (JavaScript)
Returns the nearest single-precision floating-point format of a number.  
  
## Syntax  
  
```  
  
Math.fround(  
number  
)   
```  
  
## Remarks  
 If `number` is `NaN`, the result is `NaN`.
