# toString Method (Set) (JavaScript)
Returns a string representation of a set.  
  
## Syntax  
  
```javascript  
setObj.toString()  
```  
  
#### Parameters  
 `setObj`  
 Required. A `Set` object.  
  
## Property Value/Return Value  
 The string representation of the set.
