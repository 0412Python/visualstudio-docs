# Math.tanh Function (JavaScript)
Returns the hyperbolic tangent of a number.  
  
## Syntax  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Remarks  
 The return value is the same as `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. If `x` is `NaN`, the result is `NaN`.
