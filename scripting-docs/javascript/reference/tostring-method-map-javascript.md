# toString Method (Map) (JavaScript)
Returns a string representation of a map.  
  
## Syntax  
  
```javascript  
mapObj.toString()  
```  
  
#### Parameters  
 `mapObj`  
 Required. A `Map` object.  
  
## Property Value/Return Value  
 The string representation of the map.
