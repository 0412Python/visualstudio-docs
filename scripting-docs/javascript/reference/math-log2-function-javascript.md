# Math.log2 Function (JavaScript)
Returns the base 2 logarithm of a number.  
  
## Syntax  
  
```  
Math.log2(x)   
```  
  
## Return Value  
 If `x` is `NaN`, this function returns `NaN`. If `x` is less than zero (0), this function returns `NaN`.
