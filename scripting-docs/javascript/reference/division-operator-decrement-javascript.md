# Division Operator (-) (JavaScript)
Divides the value of two expressions.  
  
## Syntax  
  
```  
  
result = number1 / number2   
```  
  
## Parameters  
 `result`  
 Any numeric variable.  
  
 `number1`  
 Any numeric expression.  
  
 `number2`  
 Any numeric expression.
