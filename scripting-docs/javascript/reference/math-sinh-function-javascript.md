# Math.sinh Function (JavaScript)
Returns the inverse hyperbolic sine of a number.  
  
## Syntax  
  
```  
  
Math.sinh(x)   
```  
  
## Remarks  
 The return value is the same as `(exp(x) - exp(-x))/2`. If x is `NaN`, the result is `NaN`.
