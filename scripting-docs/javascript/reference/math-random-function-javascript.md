# Math.random Function (JavaScript)
Returns a pseudorandom number between 0 and 1.  
  
## Syntax  
  
```  
  
Math.random(Â )  
  
```  
  
## Remarks  
 The pseudorandom number generated is from 0 (inclusive) to 1 (exclusive), that is, the returned number can be zero, but it will always be less than one. The random number generator is seeded automatically when [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] is first loaded.
