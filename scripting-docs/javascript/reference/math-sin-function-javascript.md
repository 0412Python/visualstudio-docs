# Math.sin Function (JavaScript)
Returns the sine of a number.  
  
## Syntax  
  
```  
  
Math.sin(radians)   
```  
  
## Remarks  
 The `radians` argument is a numeric expression that contains an angle measured in radians.  
  
 The return value is the sine of the numeric argument.
