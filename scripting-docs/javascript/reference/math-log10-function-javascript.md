# Math.log10 Function (JavaScript)
Returns the base 10 logarithm of a number.  
  
## Syntax  
  
```  
Math.log10(x)   
```  
  
## Return Value  
 If `x` is `NaN`, this function returns `NaN`. If `x` is less than zero (0), this function returns `NaN`.
