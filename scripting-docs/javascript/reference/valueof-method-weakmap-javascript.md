# valueOf Method (WeakMap) (JavaScript)
Returns the primitive value of the specified object.  
  
## Syntax  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### Parameters  
 `weakmapObj`  
 Required. A `WeakMap` object.  
  
## Property Value/Return Value  
 Returns the `WeakMap`instance.
