# toString Method (WeakMap) (JavaScript)
Returns a string representation of a `WeakMap` object.  
  
## Syntax  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### Parameters  
 `weakmapObj`  
 Required. A `WeakMap` object.  
  
## Property Value/Return Value  
 The string representation of the `WeakMap`.
