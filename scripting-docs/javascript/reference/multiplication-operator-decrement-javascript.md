# Multiplication Operator (*) (JavaScript)
Multiplies the value of two expressions.  
  
## Syntax  
  
```  
  
result = number1*number2  
```  
  
## Parameters  
 *result*  
 Any variable.  
  
 `number1`  
 Any expression.  
  
 `number2`  
 Any expression.
