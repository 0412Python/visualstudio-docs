# valueOf Method (Map) (JavaScript)
Returns the primitive value of the specified object.  
  
## Syntax  
  
```javascript  
mapObj.valueOf()  
```  
  
#### Parameters  
 `mapObj`  
 Required. A `Map` object.  
  
## Property Value/Return Value  
 Returns the map instance.
