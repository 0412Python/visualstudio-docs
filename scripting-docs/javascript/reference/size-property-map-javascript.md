# size Property (Map) (JavaScript)
Returns the number of elements in a map.  
  
## Syntax  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### Parameters  
 `sizeVar`  
 Required. Any number.  
  
 `mapObj`  
 Required. Any `Map` object.
