# valueOf Method (Set) (JavaScript)
Returns the primitive value of the specified object.  
  
## Syntax  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### Parameters  
 `setObj`  
 Required. A `Set` object.  
  
## Property Value/Return Value  
 Returns the set instance.
