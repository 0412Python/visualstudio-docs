# Math.sqrt Function (JavaScript)
Returns the square root of a number.  
  
## Syntax  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Remarks  
 The required `number` argument is a numeric expression.  
  
 If `number` is negative, the return value is `NaN`.
