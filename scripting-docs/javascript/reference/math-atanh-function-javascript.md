# Math.atanh Function (JavaScript)
Returns the inverse hyperbolic tangent of a number.  
  
## Syntax  
  
```  
  
Math.atanh(x)   
```  
  
## Remarks  
 If `x` is `NaN`, the result is `NaN`.  
  
 **Applies To**: [Math Object](../../javascript/reference/math-object-javascript.md)
