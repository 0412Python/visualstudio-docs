# size Property (Set) (JavaScript)
Returns the number of elements in a set.  
  
## Syntax  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### Parameters  
 `sizeVar`  
 Required. Any number.  
  
 `setObj`  
 Required. Any `Set` object.
