# ClangCompile Task

Wraps the Visual C++ compiler tool, clang.exe.

## Methods

The following tables describe the methods of the **ClangCompile** task.

### Protected Override

All methods with the `protected override` specifier:

|Method|Description|
|---------------|-----------------|
|**AlwaysAppend**|(**string**) Default is `-c`.|
|**CommandTLogName**|(**string**)|
|**ExecuteTool**|(**int**) Parameters are **string** pathToTool, **string** responseFileCommands, **string** commandLineCommands. |
|**GenerateResponseFileCommandsExceptSwitches**|(**string**) Parameter are **string[]** switchesToRemove, **CommandLineFormat** format [default **CommandLineFormat.ForBuildLog**], **EscapeFormat** escapeFormat [default **EscapeFormat.EscapeTrailingSlash**].|
|**MaintainCompositeRootingMarkers**|(**bool**)|
|**ReadTLogNames**|(**string[]**)|
|**RemoveTaskSpecificInputs**|(**void**) Parameter is **CanonicalTrackedInputFiles** compactInputs.|
|**ResponseFileEncoding**|(**Encoding**)|
|**StandardOutputEncoding**|(**Encoding**)|
|**StandardErrorEncoding**|(**Encoding**)|
|**SwitchOrderList**|(**ArrayList**)|
|**ToolName**|(**string**) Default is *clang.exe*.|
|**TrackerImmediateDirectory**|(**string**)|
|**TrackedInputFiles**|(**ITaskItem[]**)|
|**WriteTLogNames**|(**string[]**)|

#### Public Virtual

All methods with the `public virtual` specifier:

|Method|Description|
|---------------|-----------------|
|**AdditionalIncludeDirectories**|(**string[]**) Specifies one or more directories to add to the include path; separate with semi-colons if more than one. (uses `-I[path]`)|
|**AdditionalOptions**|(**string**) Additional Options.|
|**BufferSecurityCheck**|(**string**) The Security Check helps detect stack-buffer over-runs, a common attempted attack upon a program's security. (uses `fstack-protector`)|
|**BuildingInIde**|(**bool**)|
|**CLanguageStandard**|(**string**) Determines the C language standard. (uses `std=[value]`)<br/><br/>Takes value of **c89**, **c99**, **c11**, **gnu99**, or **gnu11**.|
|**CompileAs**|(**string**) Select compile language option for .c and .cpp files. 'Default' will detect based on .c or .cpp extention. (uses `-x c`, `-x c++`)|
|**CppLanguageStandard**|(**string**) Determines the C++ language standard. (uses `std=[value]`)<br/><br/>Takes value of **c++98**, **c++11**, **c++1y**, **gnu++98**, **gnu++11**, or **gnu++1y**.|
|**DataLevelLinking**|(**bool**) Enables linker optimizations to remove unused data by emitting each data item in a separate section.|
|**DebugInformationFormat**|(**string**) Specifies the type of debugging information generated by the compiler.<br/><br/> **None**, produces no debugging information, so compilation may be faster. (uses `g0`)<br/>**FullDebug**, generate DWARF2 debug information. (uses `g2 -gdwarf-2`)<br/>**LineNumber**, generate Line Number information only. (uses `gline-tables-only`)|
|**EnableNeonCodegen**|(**bool**) Enables code generation for NEON floating point hardware. This is applicable for arm architecture only.|
|**ExceptionHandling**|(**string**) Specifies the model of exception handling to be used by the compiler.<br/><br/>**Disabled** (uses `fno-exceptions`) disable exception handling.<br/>**Enabled** (uses `fexceptions`), enable exception handling.<br/>**UnwindTables** (uses `funwind-tables`), generates any needed static data, but does not affect the code generated.|
|**FloatABI**|(**string**) Selection option to choose the floating point ABI.<br/><br/>**soft** (uses `mfloat-abi=soft`), *Soft* causes compiler to generate output containing library calls for floating-point operations.<br/>**softfp** (uses `mfloat-abi=softfp`), *SoftFP* allows the generation of code using hardware floating-point instructions, but still uses the soft-float calling conventions.<br/>**hard** (uses `mfloat-abi=hard`), *Hard* alows generation of floating-point instructions and uses FPU-specific calling conventions.|
|**ForcedIncludeFiles**|(**string[]**) One or more forced include files. (uses `-include [name]`)|
|**FunctionLevelLinking**|(**bool**) Allows the compiler to package individual functions in the form of packaged functions (COMDATs). Required for edit and continue to work. (uses `ffunction-sections`)|
|**GccToolChain**|(**string**) Folder path to Gcc Tool Chain.|
|**MSCompatibility**|(**bool**) Enable full Microsoft Visual C++ compatibility.|
|**MSCompatibilityVersion**|(**string**) Dot-separated value representing the Microsoft compiler version number to report in _MSC_VER (0 = don't define it (default)).|
|**MSExtensions**|(**bool**) Accept some non-standard constructs supported by the Microsoft compiler.|
|**MSCompilerVersion**|(**string**) Microsoft compiler version number to report in _MSC_VER (0 = don't define it (default)).|
|**MSVCErrorReport**|(**bool**) Report errors which Visual Studio can use to parse for file and line information.|
|**ObjectFileName**|(**string**) Specifies a name to override the default object file name; can be file or directory name. (uses `/Fo[name]`)|
|**OmitFramePointers**|(**bool**) Suppresses creation of frame pointers on the call stack.|
|**Optimization**|(**string**) Specifies the optimization level for the application.<br/><br/>**Custom**, custom optimization.<br/>**Disabled** (uses `O0`), disable optimization.<br/>**MinSize** (uses `Os`), optimize for size.<br/>**MaxSpeed** (uses `O2`), optimize for speed.<br/>**Full** (uses `O3`), expensive optimizations.|
|**PositionIndependentCode**|(**bool**) Generate Position Independent Code (PIC) for use in a shared library.|
|**PrecompiledHeader**|(**string**) Enables creation or use of a precompiled header during the build.|
|**PrecompiledHeaderFile**|(**string**) Specifies header file name to use for precompiled header file. This file will be also added to **Forced Include Files** during build.|
|**PrecompiledHeaderOutputFileDirectory**|(**string**) Specifies the directory for the generated precompiled header. This directory will be also added to **Additional Include Directories** during build.|
|**PrecompiledHeaderCompileAs**|(**string**) Select compile language option for precompiled header file (uses `-x c-header`, `-x c++-header`)|
|**PreprocessorDefinitions**|(**string[]**) Defines a preprocessing symbols for your source file. (uses `-D`)|
|**RuntimeLibrary**|(**string**) Specify runtime library for linking. (uses `MSVC /MT`, `/MTd`, `/MD`, `/MDd` switches)<br/><br/>**MultiThreaded**, causes your application to use the multithread, static version of the run-time library.<br/>**MultiThreadedDebug**, defines _DEBUG and _MT. This option also causes the compiler to place the library name LIBCMTD.lib into the .obj file so that the linker will use LIBCMTD.lib to resolve external symbols.<br/>**MultiThreadedDLL**, causes your application to use the multithread- and DLL-specific version of the run-time library. Defines _MT and _DLL and causes the compiler to place the library name MSVCRT.lib into the .obj file.<br/>**MultiThreadedDebugDLL**, defines _DEBUG, _MT, and _DLL and causes your application to use the debug multithread- and DLL-specific version of the run-time library. It also causes the compiler to place the library name MSVCRTD.lib into the .obj file.|
|**RuntimeTypeInfo**|(**bool**) Adds code for checking C++ object types at run time (runtime type information). (uses `frtti`, `fno-rtti`)|
|**ShowIncludes**|(**bool**) Generates a list of include files with compiler output. (uses `-H`)|
|**Sources**|(**ITaskItem[]**) *required*|
|**StrictAliasing**|(**bool**) Assume the strictest aliasing rules. An object of one type will never be assumed to reside at the same address as an object of a different type.|
|**Sysroot**|(**string**) Folder path to the root directory for headers and libraries.|
|**TargetArch**|(**string**) Target Architecture.|
|**ThumbMode**|(**string**) Generate code that executes for thumb microarchitecture. This is applicable for arm architecture only.<br/><br/>**Thumb** (uses `mthumb`), generate Thumb code.<br/>**ARM** (uses `marm`), generate Arm code.<br/>**Disabled**, option not applicable for chosen platform.|
|**TrackerLogDirectory**|(**string**) Tracker Log Directory.|
|**TreatWarningAsError**|(**bool**) Treats all compiler warnings as errors. For a new project, it may be best to use `/WX` in all compilations; resolving all warnings will ensure the fewest possible hard-to-find code defects.|
|**UndefinePreprocessorDefinitions**|(**string[]**) Specifies one or more preprocessor undefines. (uses `-U [macro]`)|
|**UndefineAllPreprocessorDefinitions**|(**bool**) Undefine all previously defined preprocessor values. (uses `-undef`)|
|**UseMultiToolTask**|(**bool**) Multi-processor Compilation.|
|**UseShortEnums**|(**bool**) Enum type uses only as many bytes required by input set of possible values.|
|**Verbose**|(**bool**) Show commands to run and use verbose output.|
|**WarningLevel**|(**string**) Select how strict you want the compiler to be about code errors. Other flags should be added directly to **Additional Options**. (uses `/w`, `/Weverything`)<br/><br/>**TurnOffAllWarnings** (uses `w`), disables all compiler warnings.<br/>**EnableAllWarnings** (uses `Wall`), enables all warnings, including those disabled by default.|
