# IDiaEnumTables::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

  
Resets an enumeration sequence to the beginning.  
  
## Syntax  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Return Value  
 Returns `S_OK`.
