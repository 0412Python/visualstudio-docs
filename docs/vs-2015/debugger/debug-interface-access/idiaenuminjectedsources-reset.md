# IDiaEnumInjectedSources::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

The latest version of this topic can be found at [IDiaEnumInjectedSources::Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenuminjectedsources-reset).  
  
Resets an enumeration sequence to the beginning.  
  
## Syntax  
  
```  
HRESULT Reset();  
```  
  
## Return Value  
 Returns `S_OK`.
