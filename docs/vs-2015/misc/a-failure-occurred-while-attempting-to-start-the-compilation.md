# A failure occurred while attempting to start the compilation
An unknown error has been encountered while issuing the **Start Build** request to the [!INCLUDE[vbprvb](../includes/vbprvb-md.md)] or [!INCLUDE[csprcs](../includes/csprcs-md.md)] compiler.  
  
 **To correct this error**  
  
-   Restart [!INCLUDE[vsprvs](../includes/vsprvs-md.md)].  
  
     This error will cause the build process to fail.
