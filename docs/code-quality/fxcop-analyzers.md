# Migrate from FxCop code analysis to .NET compiler platform analyzers

Binary code analysis for managed code will be deprecated in the next major version of Visual Studio. The tool that's being deprecated is commonly known as FxCop. This page is designed to help you learn how to migrate from FxCop to the new way of analyzing managed code using analyzers.
