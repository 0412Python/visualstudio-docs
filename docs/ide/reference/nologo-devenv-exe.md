# /NoLogo (devenv.exe)

Prevents copyright information from being shown.

## Syntax

```shell
devenv /NoLogo
```

## Remarks

This switch can be useful with command-line builds and tools that inspect the output of the build.

## Example

```shell
devenv /nologo
```
